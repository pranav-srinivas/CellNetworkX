Help and Instructions

Welcome to Protein X! Protein X is a software that allows biologists to view data in multiple representations. You can either load your own data or use the data already loaded in the program. Once you have done this you should select how you want to view your data by clicking background network and then show network. From here you should be able to see many of the proteins involved in different cancers. The small circles represent different proteins and the larger circles represent different cancers. The different colored arrows also signify certain types of transfers. The green arrow represents activation, and the blue arrows represent expression.Clicking on one of the larger circles will take you to a new screen where you will see all the proteins and pathways involved in that cancer. At this point if you have your own expression data you should upload it by clicking on the tab “Expression Data”. Once you have done this you can click on the button “Expression Charts”, and will be able to see a line or bubble chart of the data shown on the screen. These shows a relation of protein levels, RNA levels, and time. The x-axis represent time and the y-axis represents protein levels. The radius of the circle represents the RNA level. These can be seen on most pathways. We hope you find this program to be useful to you if not now, then in the near future. Sonia Sachar, Nitya Kasturi, Neha Srivastava, Pranav Srinivas
